Apigee logo Extension

Apigee

<span class="apigee-logo"><span>Api</span>gee</span>

If the logo webfont can't be loaded it will display "Apigee" with the default font.

The webfont splits the logo into two parts to properly apply colors to each part: api at the characters A and a, and gee at the characters G and g. The characters P, p, I, i, E, e are blank to make the logo properly reproducible when writing "Apigee" with the font.

Write only the whole words "Apigee", "api", or "gee" (case insensitive) when using the webfont, not the individual characters — this makes your document readable when stylesheets/webfonts are disabled or cannot be used (e.g. screenreaders), and indexable by search engines for SEO.

Navbar

Use the navbar-brand-apigee class to add the Apigee logo to the navbar brand (see examples below).

Don't forget to keep the "Apigee" as text inside the brand to be SEO/screenreader friendly. Hide the "Apigee" text with the text-hide class — using the standard apigee-logo markup works, but is not necessary. If you want to add additional text inside navbar-brand, wrap it in a span and apply the navbar-brand-text class to it for proper padding. Separate "Apigee" and the additional brand text with a zero-width space (&#8203;).
